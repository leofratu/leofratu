## Highlights

### [llm-sleep](https://github.com/leofratu/llm-sleep) — Dream emulator for AI agents
Biologically-inspired sleep stages for LLM agents: replay, clustering, compression, nightmares, and morning reflection. Each stage is a separate LLM call with its own temperature and prompt, routable to any provider (Anthropic, Google, OpenAI-compatible). Treats agent downtime as structured offline compute.

### [poly-alpha](https://github.com/leofratu/poly-alpha) — Quantitative prediction market engine
Professional quant strategy runner for Polymarket. Implements Shin debiasing, tail-risk modelling via Gemini, and autonomous execution. Built for alpha generation on prediction markets.

### [nootstudy](https://github.com/leofratu/nootstudy) — iOS spaced-repetition platform
SwiftData-backed learning app with a Gemini ARIA AI companion. Designed around the science of spaced repetition for IB and high-school students. 2 stars, actively maintained.
